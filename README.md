# Exp-1 ATM & BANKING SYSTEM

# AIM:
To design and develop an ATM & Banking System using Software Engineering concepts to perform basic operations like balance check, cash withdrawal, deposit, and user authentication.

# SRS (Procedure):

1. Identify the users like customer, admin, and bank server.
2. Collect requirements such as withdraw, deposit, balance check, and PIN verification.
3. Define functional requirements (what the system should do).
4. Define non-functional requirements like security, speed, and reliability.
5. Identify inputs (card details, PIN, amount) and outputs (cash, receipt, messages).
6. Prepare the SRS document clearly with all details.
